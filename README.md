# Банковская система

## Описание проекта
Веб-приложение для управления банковскими счетами и переводами денег.

## Автор
**Виктория Прохорова**  
Группа: РГЗ-2024

## Технологии
- Backend: Python Flask + SQLAlchemy
- Frontend: HTML5, CSS3, JavaScript
- Database: SQLite
- API: JSON-RPC

## Функционал
- Авторизация клиентов и менеджеров
- Переводы между счетами
- Просмотр истории операций
- Управление пользователями (для менеджеров)
